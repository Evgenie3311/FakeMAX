# FakeMAX
Приложение-копия MAX, которое не открывается.

# Описание
Это просто приложение где нет ничего кроме логотипа MAX, имени приложения и его названия пакета.

# Предназначение
Для тех, кого заставляют специально скачать ненужный мессенджер, как было например со Сферумом.

# Опен-сурс где?
Потерян со временем. Если так боишься вирусов - иди декомпилируй самостоятельно, там лишь finishAffinity(); блок в коде. Обфускации нет.

# Установка
Распакуйте архив через любой проводник и установите APK-файл.
